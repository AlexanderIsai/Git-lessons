 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
Если мы предполагаем, что гитбаш у нас установлен. Тогда (для виндоус):
- на гитхабе заходим в нужный репозиторий
- жмем кнопку "код", выбираем SSH и копируем ссылку
- создаем/открываем нужную папку, из которой мы будем работатьс нашим репозиторием
- жмем там правым кликом мышки и выбираем Git Bash Here (чтобы не прыгать по папкам в командной строке)
- в командной строке пишем: git clone + скопированный адрес с гитхаба


Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
Мы снова предполагаем, что каталог с нужными файлами уже создан на локальном компьютере.
Тогда:
- открываем нужную папку (локальный репозиторий)
- жмем там правым кликом мышки и выбираем Git Bash Here
- пишем git init 
- пишем git add . (если хотим добавить все файлы из каталога...если не все - перечислям)
- пишем git commit -m "name of commit"
идем на гитхаб, создаем новый пустой репозиторий (через кнопку + "Create new")
копируем нужную команду из подсказок (git remote add origin git@githab...) и вставляем ее в командную строку - связываем локальный репозиторий с гитхабом
копируем следующую команду для изменения имени главной ветки (git branch -M main)
пушим наши закомиченные файлы на репозиторий в первый раз через команду git push -u origin main

Как получить на локальный компьютер изменения с github.com
Мы предполагаем, что наш локальный репозиторий связан с репозиторием на Гитхабе, тогда:
- жмем там правым кликом мышки и выбираем Git Bash Here
- пишем git pull 


Какую информацию нам дают команда :
git status
Дает информацию о текущем состоянии нашего локального репозитория. Можем видеть информацию о:
неотслеживаемых файлах, файлах которые были изменены, но не добавлены, файлах, которые еще не закоммиченны
ну и на какой ветке мы находимся
git log
Дает информацию об истории коммитов в репозитории (в хронологическом порядке, начиная с последнего), автора (с эл.адресом) и даты коммитов
и сообщения коммитов
git remote -v 
Показывает связь с удаленным репозиторием
